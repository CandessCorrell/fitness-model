### Express Node.js API Server

Clone, install, start.
```
git clone https://github.com/private-pilot/fitness-model/
cd fitness-model/express-server
npm install
bash pm2-begin.sh
```

To stop
```
pm2 stop www
```

To tail the logs generated by the server
```
pm2 logs www
```
